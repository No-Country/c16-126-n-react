User stories

Proyecto: Aplicacion de servicios profesionales

Tenemos dos tipos de usuarios: Cliente || Profesional


* Como Cliente, quiero poder registrarme, para ver la lista de profesionales en mi ciudad

* Como Profesional, quiero poder cargar mi servicio para que pueda visualizarme un posible Cliente

* Como Cliente quiero poder filtrar entre categarias para poder buscar con mas facilidad

* Como Profesional quiero poder cargar imagenes de mis trabajos para generar buenas primeras impresiones

* Como Cliente quiero poder contratar a un Profesional para que pueda resolver mi problema

* Como Cliente quiero poder cargar reseñas de Profesionales contratados para poder realizar criticas para futuros clientes

* Como Cliente quiero póder ver criticas previas para tener una referencia del Profesional

* Como Profesional quiero poder escribir reseñas sobre un Cliente para realizar criticas para futuros Profesionales

* Como Profesional quiero poder ver critcas previas para poder decidir si trabajo con un Cliente o no

* Como Cliente quiero poder realizar pagos a traves de la aplicación

* Como Profesional quiero poder realizar cobro a traves de la aplicación
